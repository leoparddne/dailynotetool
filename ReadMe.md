## 最佳实践
1. 当日工作开始时打开程序
2. 完成一项任务或中断需要处理其他事务时添加记录
3. 当日结束工作后点击复制
4. 记录填写错误后可撤销最后一条数据随后重新编写
```
复制的内容将包含当日工作总时长以及各项任务的时长
```


## 其他
